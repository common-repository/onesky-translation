== Description ==

OneSky Translation makes it easy to build multilingual WordPress sites using human translation.

With OneSky Translation, you get:

* One step Easy Translation
1. Simply pick the content to translate on your dashboard
  - Orders would be picked up by our professional translators 
2. Directly publish your translated content with just 1 click

* Powerful Features for Easy Multilingual Management:
1. Easy Translation Management
  - 1-click content selection for translation
  - 1-click easy-publishing for translated content
  - Support 30+ languages
  - Search posts according to languages
2. User-friendly Visitor Interface
  - Beautiful language switcher widget (for visitors to choose languages)
  - Option to display only translated content according to visitorâ€™s language
3. Improve SEO
  - Translations are cached and would be indexed by search engines as unique documents
4. Easy Install and Uninstall
  - The database contents for posts remain unmodified 
5. Full Customer Support
  - Any enquiry would be answered within 24 hours 

== Installation ==

1. Install OneSky Translation either via the WordPress.org plugin directory, or by uploading the files to your server
2. After activating the plugin, go to OneSky API Config in admin page to setup the OneSky account.
3. That's it. You're ready to go!
